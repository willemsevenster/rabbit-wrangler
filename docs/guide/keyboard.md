# Keyboard Shortcuts

Rabbit Wrangler is built to drive from the keyboard. Here's the full reference.

## The sidebar tree

| Key | Action |
| --- | --- |
| **Up / Down** | Move the highlight between rows |
| **Left / Right** | Collapse/expand a row, or jump to its parent/child |
| **Enter** or **Space** | Open or activate the highlighted item |
| **Home / End** | Jump to the first / last item |

## The message list

| Key | Action |
| --- | --- |
| **Up / Down** | Change the selected message |
| **Enter** | Jump into the payload |

## Moving between regions

| Key | Action |
| --- | --- |
| **F6** | Cycle focus forward through the major regions: tree → active tab → message list → payload |
| **Shift+F6** | Cycle focus in reverse |

As focus moves, the region that gains focus is briefly highlighted so you can see where you are.

## Tabs

| Key | Action |
| --- | --- |
| **Ctrl+Tab** | Move to the next open tab (wraps around at the end) |
| **Ctrl+Shift+Tab** | Move to the previous open tab (wraps around at the start) |

## Search

| Key | Action |
| --- | --- |
| **Ctrl+F** (**Cmd+F** on macOS) | Open the search popup |

See [Search](./search) for what it does.

## Bulk tab actions

::: tip
**Right-click the QUEUES group** in the tree to **open or close all of that connection's queue tabs at once** — handy for clearing your workspace or reopening everything you were looking at.
:::
